# Procedure — fact.check

procedure (no new protocol intent).

Inputs

- Claim to verify, location reference, tolerance for uncertainty.

Preconditions

- TU open; claim framed in surface language.

Process

1. Identify authoritative sources; collect 2–3 citations.
2. Compare consistency; note disputes; assign provisional posture.
3. Provide neutral phrasing and implications; list risks and mitigations.
4. Emit `tu.checkpoint` summarizing findings and recommended action.

Outputs

- Checkpoint with findings; if needed, `research_memo` for fuller treatment.
