# Policy and Routing Examples

Examples for configuring policies and tier routing.

## Examples

- `10_policy_basic.py` - Basic policy configuration
- `11_policy_custom.py` - Custom policy implementation
- `12_policy_serialization.py` - Save/load policies

## Concepts

- TTL configuration
- Capacity limits
- Promotion/demotion thresholds
- Tier selection logic

## Quick Start

```bash
python examples/policies-and-routing/10_policy_basic.py
```
