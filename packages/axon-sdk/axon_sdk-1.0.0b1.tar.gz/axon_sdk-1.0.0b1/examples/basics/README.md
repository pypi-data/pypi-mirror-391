# Basic Examples

Foundational examples for getting started with Axon.

## Examples

- `basic_usage.py` - Simple store and recall operations
- `memory_system_basic.py` - MemorySystem basics
- `memory_system_chatbot.py` - Basic chatbot implementation
- `memory_system_knowledge_base.py` - Knowledge base example
- `adapter_usage.py` - Working with storage adapters
- `embedder_examples.py` - Using different embedders
- `chroma_knowledge_assistant.py` - ChromaDB knowledge assistant

## Prerequisites

Most examples work with the development configuration (no external dependencies).

## Quick Start

```bash
# Run a basic example
python examples/basics/basic_usage.py
```
